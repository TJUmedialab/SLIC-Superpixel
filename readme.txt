1. type 'help mexGenerateSuperPixel' in the command window for the detail usage of 
    the function mexGenerateSuperPixel

2. demo.m show the usage of mexGenerateSuperPixel

3. To compile it, type 
   '>> mex mexGenerateSuperPixel\main.cpp mexGenerateSuperPixel\SLIC.cpp mexGenerateSuperPixel\SLICMask.cpp OPTIMFLAGS="/openmp $OPTIMFLAGS" -output mexGenerateSuperPixel'
   in the command window or open 'mexGenerateSuperPixel.sln' in visual studio
